PASO 1: EJECUTAR SQL (2 min)
────────────────────────────
mysql -u usuario -p base_datos < agregar_campos_ofertas_mejoradas.sql

O desde phpMyAdmin:
- Abre phpMyAdmin
- Selecciona tu base de datos
- Pestaña "SQL"
- Copia y pega el contenido del archivo
- Click "Continuar"


PASO 2: VERIFICAR ARCHIVOS (1 min)
───────────────────────────────────
✓ ofertas.php
✓ mis_ofertas.php
✓ css/ofertas-styles.css


PASO 3: PROBAR (2 min)
───────────────────────
1. Accede como vendedor Premium
2. Ve a "Mis Ofertas"
3. Crea una oferta de prueba
4. Visita ofertas.php

   → Verifica que JavaScript esté habilitado
   → Prueba en HTTPS
